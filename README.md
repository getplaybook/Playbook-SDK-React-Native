# react-native-playbook-sdk

Playbook SDK Wrapper

## Installation

```sh
npm install react-native-playbook-sdk
```

## Usage

```js
import PlaybookSdk from "react-native-playbook-sdk";

// ...

const result = await PlaybookSdk.multiply(3, 7);
```
